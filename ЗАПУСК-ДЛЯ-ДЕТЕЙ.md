# 🌿 Как запустить «Зелёное образование» на своём компьютере

**Для кого эта инструкция:** если тебе 10–16 лет (или ты просто хочешь пошагово «куда нажимать»).

**Что получится в конце:** сайт откроется в браузере по адресу **http://localhost:3000** — можно смотреть курсы, входить в аккаунт и учиться.

**Сколько времени:** первый раз — **20–40 минут** (скачиваются программы и файлы). Потом — **1–2 минуты** на каждый запуск.

---

## 📦 Шаг 0. Распакуй архив

1. Скачай файл **`GreenEdu-latest.zip`** (или как он называется у тебя).
2. **Правой кнопкой** по архиву → **«Извлечь всё…»** / **Extract All**.
3. Положи папку, например, сюда:  
   `C:\Projects\GreenEdu`  
   (можно любую папку, главное — **запомни, где она**).

Внутри должны быть файлы:
- **`ЗАПУСК.bat`** ← главная кнопка запуска
- **`ЗАПУСК-ДЛЯ-ДЕТЕЙ.md`** ← эта инструкция
- папки `src`, `public`, `scripts`, `prisma` и другие

---

## ① Установи Node.js (один раз)

**Node.js** — программа, которая запускает наш сайт.

1. Открой в браузере: **https://nodejs.org/**
2. Нажми большую зелёную кнопку **«LTS»** (Recommended For Most Users) → **Download**.
3. Запусти скачанный файл (например `node-v22.x.x-x64.msi`).
4. В установщике:
   - жми **Next** → **Next** → …
   - галочку **«Automatically install necessary tools»** можно оставить
   - в конце **Install** → **Finish**
5. **Перезагрузи компьютер** (или хотя бы закрой и открой все окна терминала).

**Проверка (необязательно):**
- Нажми **Win + R**, напиши `cmd`, Enter.
- В чёрном окне напиши: `node -v` и Enter.
- Должно показать что-то вроде `v22.x.x` — значит всё ок.

---

## ② Установи Docker Desktop (один раз)

**Docker** — «коробка» с базой данных. Сайту нужна база, чтобы хранить логины и пароли.

1. Открой: **https://www.docker.com/products/docker-desktop/**
2. Нажми **Download for Windows** (или для Mac, если у тебя Mac).
3. Запусти установщик **Docker Desktop Installer.exe**.
4. В установщике:
   - согласись с лицензией → **OK**
   - оставь галочку **Use WSL 2** (если есть) → **OK**
   - дождись конца → **Close and restart** (может попросить перезагрузку)
5. После перезагрузки **запусти Docker Desktop** из меню Пуск (иконка кита 🐳).
6. Подожди 1–3 минуты, пока внизу не будет написано что Docker **running** / **Engine running**.
   - Если спросит про **Terms** — нажми **Accept**.
   - Если спросит про регистрацию — можно **Skip** / пропустить.

**Важно:** перед каждым запуском сайта Docker Desktop должен быть **включён** (иконка кита в трее справа внизу).

**Проверка (необязательно):**
- В `cmd` напиши: `docker --version` → должна показаться версия.

---

## ③ Запусти сайт (каждый раз, когда хочешь поучиться)

### Самый простой способ — двойной клик

1. Убедись, что **Docker Desktop запущен** (кит в трее).
2. Открой папку проекта (например `C:\Projects\GreenEdu`).
3. **Дважды щёлкни** по файлу **`ЗАПУСК.bat`**.
4. Появится **чёрное окно** — **не закрывай его**, пока учишься! Это «двигатель» сайта.
5. Подожди. В первый раз может занять **5–15 минут**:
   - скачается база данных (~100 МБ)
   - установятся пакеты npm
   - создастся демо-аккаунт
6. Когда всё готово — **сам откроется браузер** с сайтом.  
   Если не открылся — сам зайди на: **http://localhost:3000**

### Что ты увидишь в чёрном окне (это нормально)

```
[1/6] Проверка Node.js...
[2/6] Копируем готовые настройки...
[3/6] Запуск PostgreSQL в Docker...
[4/6] Ждём готовности PostgreSQL...
[5/6] Установка зависимостей...
[6/6] Создание таблиц и демо-пользователя...
Запускаем сайт на http://localhost:3000 ...
```

## ④ Войди на сайт

### Вариант А — готовый демо-аккаунт (проще всего)

1. Нажми **«Логин»** (или **«Вход»** на телефоне).
2. Введи:

| | |
|---|---|
| **Email** | `demo@demo.ru` |
| **Пароль** | `demo1234` |

3. Нажми войти → в меню появится **«Курсы»**.

### Вариант Б — свой аккаунт

1. Нажми **«Регистрация»**.
2. Придумай email и пароль (**минимум 6 символов**).
3. Войди и переходи к курсам.

---

## ⑤ Как пользоваться сайтом

1. **Главная** — кнопка «Начнём учиться!» или «Перейти к курсам».
2. **Курсы** — список из 4 тем; нажми на карточку.
3. Внутри курса — **видео**, **лекции (PDF)**, **тесты** (ссылки на Google Forms).
4. Кнопка **«← Назад»** — вернуться на предыдущую страницу.
5. Внизу может появиться плашка **про cookie** — нажми **«Принять»**.

---

## ⑥ Как остановить сайт

1. Перейди в **чёрное окно** (где бежит текст).
2. Нажми **`Ctrl + C`** на клавиатуре  
   **или** просто **закрой** это окно крестиком.
3. Docker можно оставить включённым или выключить через Docker Desktop → **Quit**.

---

## 🔁 Запуск на следующий день

1. Запусти **Docker Desktop** → подожди, пока «running».
2. Снова **дважды щёлкни `ЗАПУСК.bat`**.
3. Открой **http://localhost:3000** (или дождись автоматического открытия).

Переустанавливать Node и Docker **не нужно**.

---

## ❗ Если что-то пошло не так

### «Node.js не установлен»
→ Вернись к **шагу ①**, установи Node LTS, перезагрузи ПК.

### «Docker не найден» / «Docker Desktop is not running»
→ Вернись к **шагу ②**, запусти Docker Desktop из Пуска, подожди 2 минуты.

### Долго крутится на первом запуске
→ **Это нормально.** Нужен интернет. Не закрывай окно 15 минут.

### Сайт не открылся сам
→ Открой браузер (Chrome / Edge) и введи в адресной строке:  
**http://localhost:3000**

### «Неверный email или пароль»
→ Используй **demo@demo.ru** / **demo1234**  
или зарегистрируйся заново.

### Порт 3000 занят
→ Закрой другие чёрные окна с сайтом или перезагрузи компьютер.

### Полный сброс (удалить всех пользователей и начать с нуля)

1. Открой папку проекта.
2. В адресной строке проводника напиши `cmd` и Enter.
3. Выполни по очереди:
   ```
   docker compose down -v
   npm run go
   ```

---

## 🖥 Запуск на Mac или Linux (без .bat)

1. Установи Node.js и Docker Desktop (как выше, но версии для Mac).
2. Открой **Терминал**.
3. Перейди в папку проекта:
   ```bash
   cd ~/Projects/GreenEdu
   ```
4. Запусти:
   ```bash
   npm run go
   ```
5. Открой **http://localhost:3000**
